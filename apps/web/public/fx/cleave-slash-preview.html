Cleave slash — frame ladder

A static mirror of pixiFx.cleaveSlash, drawn in canvas 2D at fixed points in its life so the look can be judged without the Pixi ticker running (which blocks screen capture). The middle grey card is the struck unit (the outer two are context). The cut is a FIXED spanPx length centred on the target — never derived from the card or the struck group — so every cleave, and the tuner's Test button, draw exactly the same rake. The hit-stop and the return delay live in the lunge timeline and cannot be shown here; only the slash and its drips can. This rig shares no code with the shipped effect — if the maths drift, it stops being evidence.
